# Homebrew Tap

This tap contains Homebrew casks for my macOS applications.

## Available Casks

| Cask | Description |
|------|-------------|
| [containerbar](https://github.com/michaeltookes/ContainerBar) | Docker container monitoring from your macOS menu bar |
| [prompter](https://github.com/michaeltookes/prompter) | macOS presenter overlay and teleprompter app |

## Installation

```bash
brew tap michaeltookes/tap
brew install --cask containerbar
brew install --cask prompter
```

Or install directly:

```bash
brew install --cask michaeltookes/tap/containerbar
brew install --cask michaeltookes/tap/prompter
```

## Updating

```bash
brew update
brew upgrade --cask containerbar
brew upgrade --cask prompter
```

## Uninstalling

```bash
brew uninstall --cask containerbar
brew uninstall --cask prompter
```
